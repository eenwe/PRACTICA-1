Se ha creado una aplicacion que es capaz de extraer informacion de varias fuentes, las cuales con un proceso de homogeneizacion logran asimilar los datos bajo los mismos parametros
    Yahoo Fiannce y Alpha advantage

Se han establecido clases y jerarquias para el calculo de distintos parametros correspondientes a los simnbolos solicitados por el ususario
  La clase REPORT dispone de datos y calculos para los valores individuales, esta tiene una clase hija que imprime los resultados
  La clase cartera dispone de calculos y datos para el conjunto de valores, similares a los de la clase report

  Dentro de las dos clases anteriores se encuentra la posibilidad de simular Monte Carlo

  Se establece una clase montecarlo, donde calcula una simulacion, asi mismo una clase hija se establece para realizar calculos en la clase Monte Carlo con distribucion t de student

  Se establece otra clase con la posibilidad de realizar una simulacion empirica

Por ultimo se ha establecido unos inputs para que el usuario solicite los datos que considere, asi como los graficos que le sean necesarios, ademas, esta incluida la posibilidad del reprocesado de datos en caso que la informacion introducida no sea valida
